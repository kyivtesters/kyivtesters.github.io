We are Typesafe Utrecht the meetup group, which welcomes everyone, who has an opinion on strong typing in programming languages. Let's go cross-lingual and discuss the basics of type systems, compare different implementations and follow the newest developments. Developers and computer scientists with all views and levels are welcome.

Our perfect programming world is the world of Total Functions where by looking at a function you can immediately be confident of what it does and doesn't do, because 95% of them don't have any side effects.

Where functions can be passed around easily, because they are first class and higher order.

Where types guide the implementation and parametric polymorphism helps to create powerful reusable abstractions.

Where good type inference helps the code to be not only rock-solid, but also beautiful.

Fantasy? No, it's functional programming. We need to embrace functional programming in typesafe languages, because it not only greatly improves development experience quality of our programs...
